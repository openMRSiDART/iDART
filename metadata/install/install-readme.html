Welcome to the Cell-Life iDart Pharmacy Stock Control Management Tool

The Cell-Life iDart (Intelligent Dispensing of Anti-Retroviral Treatment) Tool has been developed in collaboration with the Desmond Tutu HIV Centre. This tool is designed to support semi-rural and remote clinics from a central Pharmacy.

This tool is an open-source project and designed to alleviate the burden of ART stock management for a pharmacist.

Please feel free to distribute this program under its license agreement. Any thoughts or comments would be greatly appreciated idart@cell-life.org.
